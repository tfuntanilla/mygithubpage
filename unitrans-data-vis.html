Visualization Project

10/24/2014

I started this project summer of 2014 to learn Javascript and data visualization. The idea of the project came from the data visualization course offered at my university, which I didn't get to take because of schedule conflicts. Since the class website was still up, I thought of doing the projects anyway, just for learning experience.

Basically, the project is about visualizing a dataset using the d3.js library. I got the dataset from the class website, and it contains one month of data. Right now, all I have is an overview of the entire dataset. Eventually, I'll have more charts visualizing data from specific stops.
