Image Processing

01/17/15

It's my second to last quarter in college, and I'm pretty much just finishing up the last couple of requirements I need to graduate. One of those requirements that I'm taking right now is a team design project course split in two quarters (started this quarter will continue to spring). It's basically a capstone course where each project team researches, proposes, implements, and validates a solution to an open-ended computer science-related design problem. The class received the project assignments last week, and my team's project in a nutshell is about developing a software that performs complex data analysis and visualization for an ongoing research (on a topic I can't disclose yet) by UC Davis scientists.

The project will also involve image processing, which I don't have prior experience with. There's an image processing course at UCD, but it hasn't been offered in years, which is kind of sad for me. I definitely would have taken it if it was offered during my time at UCD. Anyway, since my team will need to do it for our project, I started searching for resources online to get myself on the right track. Fortunately, one of the first things that came up on my search is a series of UCD lecture videos for the image processing course they last offered way back when.

I've only watched the first lecture, and I'm already really interested in where these video lectures would take me. I'll update this blog post as I learn more about image processing by watching these videos and doing some programming exercises. You can check out the videos through this Youtube playlist. I'm also taking notes on a Google doc if you're interested.
