    3D Viewer

    3D Viewer is a desktop program that allows users to create a 3D scene that they can manipulate or navigate into. The scene models/objects are loaded from an .obj or a JSON file, and the application renders the objects. The user can transform each individual object (translate, rotate, or scale), manipulate the lighting and the type of shading (flat, Gouraud, or Phong) of the scene, and add textures for a more realistic rendering. The user can save a static scene in JSON format and load the same scene at a later time. Additionally, a basic implementation of the graphics pipeline is provided, giving the user the option to render on software.

    Languages/Tools: C++, OpenGL, Qt
    Source code and disk image file (OS X installer) on Github soon.


    Visualization of Directional Blood Vessels

    This Windows desktop software was developed to assist scientists and researchers from the Department of Dermatology at the UC Davis Institute for Regenerative Cures in their study of blood vessel formation. The software performs medical image processing, quantification, analysis, and visualization.

    Languages/Tools: C++, OpenCV, Qt
    Source code and executable (Windows installer) currently unavailable for public access due to the confidentiality of the client's research.
