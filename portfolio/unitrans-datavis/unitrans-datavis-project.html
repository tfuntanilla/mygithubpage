Visualization Project Update

01/09/2015

This project is pretty much completed. I don't want to launch the whole thing yet since I'm not quite satisfied with its UI design. I'll fix that eventually, but it will probably take a while since I got so many things going on right now. I've been super busy since senior year started, and since this is just a personal project, I only get to allocate time for it when I absolutely don't have anything else to do. However, after fall quarter started I almost always had other things to do so it took me so long to update the status of this project.

Anyway, I thought this project was relatively difficult, mainly because of the huge learning curve that came with learning how to use the D3 library. If I had taken the class, I could see myself spending just as many hours as I did for my computer graphics class (taught by the same professor). It was rewarding in the end though, as expected. Data visualization is fun because like computer graphics, you get to see interesting and pretty things pop up on your screen.



Visualization Project

10/24/2014

I started this project towards the end of summer of 2014 to learn JavaScript and data visualization. The idea of the project came from the data visualization course offered at my university, which I didn't get the chance to take because of schedule conflicts. Since the class website was still up, I thought of doing the projects anyway, just for learning experience.

Basically, the project is about visualizing a dataset using primarily the d3.js library. I got the dataset from the class website, and it contains one month of data. Right now, all I have is an overview of the entire dataset. Eventually, I'll have more interactive charts that go into more specific details regarding the dataset. I chose the most straightforward interactive chart for now, but that might change later on when I get better at using the D3 library.
