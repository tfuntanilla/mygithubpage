OpenGL Project

10/24/2014

This project is for the computer graphics course I am taking this quarter. It's the first phase of a quarter-long project, so right now it's just a basic 3D viewer. Still, I spend HOURS on this phase of the project. I didn't know anything about OpenGL prior to taking this class so there was such a huge learning curve. I enjoyed it though.

How the program works is it loads a scene from a JSON file, which has information about the objects (saved in .obj format), and then renders those object. There's a camera that allows one to navigate the scene using mouse countrols, and then one can also select and transform individual objects.
