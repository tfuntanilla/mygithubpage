OpenGL Project Phase 1

10/24/2014

This project is for the computer graphics course I am taking this quarter. It's the first phase of a quarter-long project, so right now it's just a basic 3D viewer. Still, I spend HOURS on this phase of the project. I didn't know anything about OpenGL prior to taking this class so there was such a huge learning curve. I enjoyed it though.

The program works by loading a scene from a JSON file, which has information about the objects (saved in .obj format), and then renders those object. There's a camera that allows you to navigate the scene using mouse countrols, and then you can also select and transform individual objects. It allows you to switch between wire frame and color-filled rendering and between perspective and orthographic projections. After making changes to the loaded scene, you can save the scene to a JSON file so you can restore it later.
